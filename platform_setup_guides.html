Platform Kurulum Rehberleri

Adım adım sosyal medya API'lerini nasıl yapılandıracağınızı öğrenin

Facebook Graph API

200 request/saat • Ücretsiz

Gerekli Alanlar:
App ID: Uygulama kimliği
App Secret: Uygulama gizli anahtarı
Access Token: Erişim token'ı
Page ID: Facebook sayfa ID'si
Facebook Developer
Kurulum Adımları:
1
Developer Hesabı Oluştur

https://developers.facebook.com adresine gidin ve "Create App" butonuna tıklayın.

2
App Tipi Seçin

Business tipini seçin ve gerekli bilgileri doldurun.

3
Facebook Login Ekleyin

"Add Product" → "Facebook Login" seçin.

4
API Bilgilerini Alın

Settings → Basic bölümünden App ID ve App Secret'ı kopyalayın.

5
Access Token Oluşturun

Tools → Graph API Explorer gidin ve "Get User Access Token" tıklayın. İzinleri seçin:

  • pages_read_engagement
  • pages_manage_posts
  • publish_to_groups
6
Page ID Bulun

Graph API Explorer'da: me/accounts sorgusunu çalıştırın.

Twitter/X API v2

500,000 tweet/ay • Ücretsiz

Gerekli Alanlar:
API Key: API anahtarı
API Secret: API gizli anahtarı
Access Token: Erişim token'ı
Access Token Secret: Erişim token gizli anahtarı
Bearer Token: Taşıyıcı token
Twitter Developer
Kurulum Adımları:
1
Developer Hesabı Oluştur

https://developer.twitter.com adresine gidin.

2
Free Account Başvurusu

"Sign up for Free Account" seçin ve formu doldurun.

3
Project Oluşturun

"Create Project" butonuna tıklayın ve project bilgilerini girin.

4
App Oluşturun

"Create App" seçin ve app ayarlarını yapın.

5
API Anahtarlarını Alın

"Keys and tokens" bölümünde:

  • API Key ve API Secret'ı kopyalayın
  • "Generate" ile Access Token ve Secret'ı oluşturun
  • Bearer Token'ı da kopyalayın
6
İzinleri Ayarlayın

App permissions'ı "Read and Write" olarak ayarlayın.

Instagram Basic Display API

200 request/saat • Ücretsiz

Gerekli Alanlar:
App ID: Facebook App ID (aynısı)
App Secret: Facebook App Secret (aynısı)
Access Token: Instagram access token
Instagram Business Account ID: Instagram işletme hesabı ID
Instagram API Docs
Kurulum Adımları:
1
Facebook Developer Kullanın

Facebook developer hesabınızı kullanın (Instagram için ayrı hesap gerekmez).

2
Instagram Basic Display Ekleyin

App'inize "Instagram Basic Display" ürününü ekleyin.

3
Platform Ayarları

"Add Platform" → "Website" seçip sitenizi girin (localhost da olabilir).

4
Test Kullanıcıları

Instagram Basic Display ayarlarından test kullanıcıları ekleyin.

5
Access Token Alın

"App Review" bölümünden access token'ı生成ın.

6
Business Account ID

Graph API Explorer'da Instagram Business Account ID'yi bulun.

YouTube Data API v3

10,000 units/gün • Ücretsiz

Gerekli Alanlar:
API Key: YouTube API anahtarı
Client ID: OAuth2 client ID
Client Secret: OAuth2 client secret
Refresh Token: OAuth2 refresh token
Channel ID: YouTube kanal ID'niz
Google Cloud Console
Kurulum Adımları:
1
Google Cloud Projesi

Google Cloud Console'da yeni proje oluşturun.

2
API'yi Etkinleştirin

"APIs & Services" → "Library" gidin ve "YouTube Data API v3" aratıp etkinleştirin.

3
OAuth2 Credentials

"APIs & Services" → "Credentials" gidin ve "OAuth 2.0 Client ID" oluşturun.

4
Application Type

"Web application" seçin ve "Authorized redirect URIs" ekleyin: http://localhost:8080/callback

5
Refresh Token Alın

OAuth2 Playground'da:

  • YouTube Data API v3 seçin
  • Authorize APIs butonuna tıklayın
  • Access token ve refresh token'ı alın
6
Channel ID Bulun

YouTube kanalınızın URL'sindeki channel ID'yi kopyalayın.

TikTok API

Business onayı gerektirir • Sınırlı

Gerekli Alanlar:
Client Key: TikTok client anahtarı
Client Secret: TikTok client gizli anahtarı
Access Token: TikTok access token
Not: TikTok API'si business account onayı gerektirir. Manuel upload daha pratik olabilir.
TikTok Developer
Kurulum Adımları:
1
Developer Hesabı

TikTok Developer hesabı oluşturun.

2
Business Account

Business account başvurusu yapın ve onay bekleyin.

3
App Oluşturun

Onay geldikten sonra app oluşturun ve Client Key/Secret alın.

4
Access Token

OAuth2 flow ile access token alın.

Alternatif: TikTok Creative Center kullanarak manuel upload daha hızlı olabilir.

İpuçları ve Yardım

İyi Pratikler:
  • Başlangıçta tek platform ile test edin
  • Rate limiting hatalarına karşı retry logic ekleyin
  • API anahtarlarını güvenli saklayın (.env dosyası)
  • Test gönderileri yaparak çalıştığından emin olun
  • Her platformun limitlerini takip edin
Sorun Giderme:
  • 401/403 hataları: API anahtarlarını kontrol edin
  • 429 hataları: Rate limit aşıldı, bekleyin
  • 500 hataları: Platform sorunu, daha sonra deneyin
  • Video upload hataları: Dosya boyutunu ve formatı kontrol edin
  • Permission hataları: İzinleri doğru ayarladığınızdan emin olun
